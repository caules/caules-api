# ✨ Caules API ✨
A open source application for copypasta sharing.

## 🛠 Requirements
- Node >=14
- Mysql
- Docker (optional)


## 💾 Installation

```bash
npm install
```

## ⚙️ Usage

```bash
node ace serve --watch
```
